kpSplitterPane

  1. - component in module angularjs-splitter

Dependencies

  • ^kpSplitter

Usage

<kp-splitter-pane
    [min-size="number"]
    [init-size="number"]
    [show]>
</kp-splitter-pane>

Arguments

Param Type Details
minSize
(optional)
 Number

Minimum size of pane in pixels.
initSize
(optional)
 Number

Initial size of pane in pixels. If not specified, panes will have width: 50%.
show
(optional)
 Boolean

If false, pane will hide. Default is true.
